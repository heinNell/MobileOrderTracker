üìã Required Setup Steps

Step 1: Deploy Database Schema

This is the most important step!

  1. Open Supabase SQL Editor:
    Open SQL Editor
  2. Copy the schema file:
    cat /workspaces/MobileOrderTracker/supabase/schema.sql

    Or view the schema file here

  3. Paste the entire contents into the SQL Editor
  4. Click "Run" button (or press Ctrl+Enter)
  5. Wait for "Success" message

Step 2: Create Admin User

  1. Go to Supabase Authentication:
    Open Auth Users
  2. Click "Add User"
  3. Enter your email and password
  4. Click "Create User"

Step 3: Add User to Database

After creating the auth user, go back to the SQL Editor and run:

-- Create tenant INSERT INTO public.tenants (name, subdomain) VALUES ('Test Company', 'test') RETURNING id; -- Create user profile (replace YOUR-EMAIL with your actual email) INSERT INTO public.users ( id, email, full_name, role, tenant_id, is_active ) VALUES ( (SELECT id FROM auth.users WHERE email = 'YOUR-EMAIL@example.com'), 'YOUR-EMAIL@example.com', 'Admin User', 'admin', (SELECT id FROM public.tenants WHERE name = 'Test Company'), true );
Open SQL Editor

Step 4: Test the System

  1. Login to Dashboard: localhost:3001
  2. Use the email and password you created
  3. Create your first order
  4. Generate a QR code
  5. Test the features!
